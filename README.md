# Chess Learning Tree

An interactive chess learning application built with Next.js.

## Features

- Interactive chess board with move visualization
- Skills profile page with d3-force bubble visualization
- Learning tree structure for chess concepts

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- Next.js 16
- React 19
- chess.js for game logic
- react-chessboard for board UI
- D3.js for visualizations
- Tailwind CSS for styling
